# 🛡️ नेत्र-कवच (Netra-Kavach) v1.0

**निर्माता:** नीरज जोशी पिथौरागढ़ (NEERAJ_JOSHI_PITHORAGARH_2026)
**मिशन:** डिजिटल इंडिया को स्कैम्स से सुरक्षित बनाना।

## 🧠 प्रोजेक्ट का परिचय (Introduction)
"नेत्र-कवच" एक अत्याधुनिक एथिकल एआई सॉफ्टवेयर है, जो आपकी प्राइवेसी से समझौता किए बिना आने वाले संदेशों का मनोवैज्ञानिक विश्लेषण करता है। यह मेरी **'जादुई डायरी' (Magical Diary)** के सिद्धांतों पर आधारित है, जो मेंटलिज्म और साइबर सुरक्षा का संगम है।

## 🚀 मुख्य विशेषताएं (Key Features)
* **इमोशन डिटेक्शन:** यह 'डर' और 'लालच' आधारित संदेशों को तुरंत पहचानता है।
* **अभेद्य सुरक्षा:** डेटा को सुरक्षित रखने के लिए **AES-256** एन्क्रिप्शन का उपयोग।
* **शून्य-डेटा पॉलिसी:** आपका डेटा केवल आपके फोन में रहता है।

## 🛠️ फाइल संरचना (Structure)
1. `main.py`: स्कैम डिटेक्शन का मुख्य इंजन।
2. `encryption.py`: डेटा को सुरक्षित रखने वाला सुरक्षा कवच।
